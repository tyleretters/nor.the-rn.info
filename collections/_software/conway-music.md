Conway Music is a grid application for Max4Live. It uses Conway's Game of Life help you create melodies and rhythms. See the thread [over at llllllll.co](http://l.llllllll.co/conway) and watch [the demo video](https://www.youtube.com/watch?v=ptIJ6MizZ88).
